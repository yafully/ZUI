ASDUI.SL.Class
调用方法：
var myshake = new ASDUI.SL(element[, options]);
一个藏袍：
	new ASDUI.SL('SL',{
		moveBox: 'SL',
		slides: '.SL_box',
		slideDuration:5000,
		effectDuration:1000,
		fadeDist:600,
		fadePosition:'left',
		stopSlideOnClick: false,
		autoSlide: true,
		navigationNums: false // if true, numbers will appear
	});
Option 参数描述
moveBox 幻灯片第二层容器Id
slides[s] 需要切换的内容容器(选择器)
slideDuration 切换的时间间隔
effectDuration 切换效果时间
effect 切换时的效果，默认为Fx.Transitions.linear
fadeDist 切换时的水平或垂直改变值；默认值为null，为null时自动获取父容器宽度高度数值作为参照；
fadePosition 切换的方向；暂时只支持left,top,fade
autoSlide 是否自动切换，默认为true
navbox 幻灯片导航容器，可为空，但是若启用容器内子元素个数必须跟幻灯片个数对等
navigationNums 是否在导航按钮中显示序号，默认为false
circle slider之间切换是否首尾连续,默认值false
autoSize 是否允许slider宽高以及切换深度随页面大小变化而变化默认值为false
onStart（fn） 幻灯片滑动前执行的回调函数
onFinish（fn） 幻灯片滑动完成后执行的回调函数
   
外置方法：.goToSlide(num);跳转到第num个幻灯片
Sl = new ASDUI.SL();	
Sl.goToSlide(3);
外置方法：.next();跳转到下一个幻灯片
Sl = new ASDUI.SL();	
Sl.next();;
外置方法：.prev();跳转到上一个幻灯片
Sl = new ASDUI.SL();	
Sl.prev();
外置方法：.prepareSlides(A);重新构建幻灯片，且默认显示第A个；本方法用于动态删除了幻灯片和resize后重新构建当前幻灯片
Sl = new ASDUI.SL();	
Sl.prepareSlides(1);
外置方法：.getCurrent();获取当前幻灯片显示的slider对象
Sl = new ASDUI.SL();	
Sl.getCurrent();//return slider object
外置方法：.getNum();获取当前幻灯片显示的slider的序列
Sl = new ASDUI.SL();	
Sl.getNum();//return number
