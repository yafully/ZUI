ASDUI.Tab.Class
调用方法：
var tab1 = new ASDUI.Tab(element,[, options]);
一个藏袍：
var tab1 = new ASDUI.Tab('demo_a');
var tab2 = new ASDUI.Tab('demo_b',{closeBtn:true});
Option 参数描述
tabSelector tab按钮的class
contentSelector tab内容容器的class
activeClass tab按钮选中时的样式
tabSbox 动态添加按钮时放按钮的容器，当使用add（）方法时不可为空。
contentbox 动态添加内容容器时的目标容器，当使用add（）方法时不可为空。
addTab 动态添加按钮的内容，使用add（）方法时有效。
addContent 动态添加容器的内容，使用add（）方法时有效。
addMax 动态添加选项卡的最大数，默认为6
closeBtn 是否允许关闭选项卡默认为false
rel（add方法私有属性） 新添加选项卡的标识，不能为空且必须唯一
exist 初始化时已经存在的选项卡个数；默认值为0；
scrollTab 是否启用标签滚动，默认值为false
step 标签每次滚动时的步长，默认值为200
onClick（Fn） 选项卡点击时触发的回调函数
onChange(Fn) 选项卡删除后触发的回调
   
   
外置方法：add({options});新增一个选项卡
var tab1 = new ASDUI.Tab('demo_a',{closeBtn:true});	
tab1.add({tabSbox:'myul',contentbox:'mypan',addTab:'1111111',addContent:'aaaaaaaaaaajj',rel:'123'});
外置方法：show(index);显示当前选项卡中第x个
var tab1 = new ASDUI.Tab('demo_a',{closeBtn:true});	

tab1.show(2);
外置方法：getIndex();获取当前激活选项卡序列号
var tab1 = new ASDUI.Tab('demo_a',{closeBtn:true});	

tab1.getIndex();
外置方法：getSecs();返回所有选项卡sec对象
var tab1 = new ASDUI.Tab('demo_a',{closeBtn:true});	

tab1.getSecs();//return array
