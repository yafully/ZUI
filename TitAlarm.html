ASDUI.TitAlarm.Class(网页标题信息提示)
调用方法：
new ASDUI.TitAlarm(options);
一个藏袍：
var st = new ASDUI.TitAlarm();
Option 参数描述
tim 闪动频率，默认值350
tit 提示的内容
delay 闪动提示时长，默认值3500
mess 全局标识，当前闪动的内容
title 原始标题
null null
外置方法：.start(string);开始提示
a = new ASDUI.TitAlarm();
a.start('[系统消息]');
